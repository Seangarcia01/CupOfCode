This is the official portfolio website for the **CupOfCode** team. It serves as a centralized hub to showcase our individual profiles, technical skills, and collaborative development work. The website is built to be fully responsive, ensuring an optimal viewing experience across desktops, tablets, and mobile devices.

## üöÄ Technologies Used

This project leverages the foundational technologies of front-end development, demonstrating our team's proficiency in core web standards and responsive design techniques.

* **HTML5:** Used for the semantic structure and content of the website.
* **CSS3:** Utilized for all styling, layout (using **Flexbox**), visual effects, and **Responsive Web Design (RWD)** via media queries.
* **JavaScript (ES6+):** Implemented for client-side interactivity, such as the navigation bar toggle on mobile screens.
* **Google Fonts:** Integrated the **DM Sans** typeface for professional typography.
